## Available Examples

The `examples/` directory contains additional examples:

| Example | Description |
|---------|-------------|
| `basic_cartesian_example.py` | Simple Cartesian path planning |
| `freespace_ompl_example.py` | OMPL freespace motion planning |
| `pick_and_place_example.py` | Pick and place with TrajOpt |
| `puzzle_piece_example.py` | Cartesian path following |
| `puzzle_piece_auxillary_axes_example.py` | 9 DOF planning with positioner |
| `glass_upright_example.py` | Constraint-based planning |
| `car_seat_example.py` | Complex assembly planning |
| `pythonic_api_example.py` | High-level Pythonic API usage |
